# 🎯 Student Marks Predictor (ML Web App)

This is a simple Machine Learning web application that predicts student marks based on study hours using Linear Regression.

- Simple and user-friendly web interface
- Fast and accurate predictions
- Accessible from mobile and desktop

## ⚙️ Installation & Setup

### 1. Clone the repository
git clone https://github.com/vivekjain060305-coder/student-marks-predictor-vivek.git

### 2. Navigate to project folder
cd student-marks-predictor

### 3. Install dependencies
pip install -r requirements.txt

### 4. Train the model
python train.py

### 5. Run the application
python app.py

### 6. Open in browser
http://127.0.0.1:10000
